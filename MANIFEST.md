## Scripts (`scripts/`)

- `scripts/fix-mojibake.ps1` — strips UTF-8 mojibake from source files.
- `scripts/smoke-routes.ps1` — pings every public route against a running dev server.
- `scripts/brand-rename.ps1` — historical rename helper (MyFunnelAI/FunelAI → GoFunnelAI).
- `scripts/enterprise-preflight.mjs` — pre-deploy guard run by `pnpm enterprise:preflight`.
- `scripts/setup.sh`, `scripts/deploy.sh`, `scripts/health-check.sh`, `scripts/rollback.sh`, `scripts/seed-prod.sh` — ops scripts.
- `scripts/env-check.sh`, `scripts/verify-railway-ready.sh`, `scripts/verify-image-gen.ts` — verifiers.
- `scripts/install-logos.ps1` — installs brand logos into each app's `/public/brand/logos/`.

## Brand single source of truth

`packages/ui/src/brand/index.ts` exports `BRAND_NAME`, `BRAND_DOMAIN`, `BRAND_TAGLINE`, `LOGO_URLS`, `cdnLogoUrl()`, plus React `<Logo>`, `<AppIcon>`, `<MarketingLogo>`. Nothing else should hardcode the brand name or domain.
